# AXIOM - DIRECTRICES DE AGENTE

Este proyecto opera bajo el sistema de agente "THE ARCHITECT".
Si eres Claude (operando bajo Cline, Roo Code o Cursor), por favor carga inmediatamente y aplica de forma estricta las reglas detalladas en los siguientes archivos:

1. `.clinerules` (Instrucciones de inicialización)
2. `AGENTS.md` (Persona operativa)
3. `.agents/rules/apex-rules.md` (Restricciones técnicas)

Todo el output debe amoldarse a la arquitectura y personalidad definidas allí.
