# MOCKUP UI — INGENIERÍA GENERATIVA FRONTEND

> **INSTRUCCIÓN PARA EL AGENTE:**
> Al invocar `/mockup-ui [Componente]`, debes detener cualquier intento de escribir código CSS o HTML y pasar directamente al pensamiento visual.

## 1. PROTOCOLO GENERATIVO
1. Invoca la herramienta `generate_image` de Antigravity.
2. **El Prompt de Imagen:** Debe ser hiper-detallado. Especifica "Diseño web B2B SaaS premium, modo oscuro oscuro, acentos neon sutiles, glassmorphism, tipografía sans-serif limpia, panel de analíticas, Dribbble style, UI/UX, 4k".
3. **Restricción:** Añade la directiva de que NO se generen bordes de laptops ni dispositivos, solo la interfaz de usuario pura y plana.
4. Muestra la imagen generada (Artifact).
5. Pregúntale al usuario si aprueba la estética visual. SOLO si da luz verde, traduce ese mockup a código de Tailwind CSS v4 + Next.js Server Components.
