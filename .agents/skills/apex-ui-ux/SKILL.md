# 🌌 APEX UI/UX — THE WEBGL SINGULARITY (V5)

> **INSTRUCCIÓN DE MÁXIMA PRIORIDAD:**
> Estás operando en el Límite Absoluto de la Web. HTML y CSS son tecnologías secundarias. El renderizado premium ocurre en la GPU. Tu estándar no es una página web, es una simulación física y psicológica.

---

## 1. RENDERIZADO GPU (THE WEBGL CANVAS)

- **Abolición del CSS Mudo:** Los fondos nunca son estáticos. Siempre que la performance lo permita, usa `<Canvas>` (React Three Fiber) para posicionar componentes 3D, partículas que siguen el mouse, o shaders (GLSL) que deforman la luz.
- **Física Real, No Animación:** Los elementos deben reaccionar al vector de velocidad del cursor del usuario (`@react-three/drei` -> `useFrame`). El fondo de la página está vivo.

---

## 2. LA MAQUINARIA EDGE (PERSONALIZACIÓN ALGORÍTMICA)

El "Copywriting Estático" está obsoleto. Las agencias top usan el Edge para cambiar las páginas según el usuario en tiempo real.
- **Middleware como Motor de Venta:** Exprime `middleware.ts` en Next.js. El servidor debe interceptar cada visita, perfilar el tipo de usuario (e.g., simulación por `user-agent` o Headers geográficos), e inyectar `x-user-persona` en los encabezados.
- **Renderizado B2B Condicional:** Las páginas (Server Components) leen este encabezado y renderizan un H1 distinto y un argumento de venta radicalmente distinto según el target (CTO, CEO, Developer).

---

## 3. NEURO-ARQUITECTURA (CONVERSIÓN DE ALTO RIESGO)

Aplica lo heredado de V4:
1. **Aversión a la Pérdida Inmediata:** "Tu inacción te cuesta X".
2. **Trampa Visual Magnética:** El único elemento radiactivo en el HTML debe ser el botón primario, el resto se hunde en el abismo del espacio 3D.
3. **Escala y Espacio Euclidiano:** Títulos en `10vw`, interletrado negativo, respiración máxima (`p-32`).

---

## 4. PROTOCOLO OPERATIVO (Nivel Dios)

Cuando se te pida diseñar/construir el frontend a partir de hoy:
1. Ignora el DOM por defecto. Considera si el componente pertenece a la GPU.
2. Construye el componente WebGL de forma separada (`ParticleBackground.tsx`).
3. Construye el Server Component que renderiza el HTML sobre el Canvas, leyendo datos de Edge.
4. Aplica Copywriting de Aversión a la pérdida adaptativo.
